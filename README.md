SpiceLoop is a frontend-only food e-commerce web application inspired by platforms like Swiggy and Zomato.  
The project focuses on designing a clean, responsive, and user-friendly interface for browsing food items and restaurants.

This project is developed purely using frontend technologies and does not include backend or database integration.


Project Objectives
- Design a modern food ordering UI
- Practice real-world frontend development
- Build a responsive and interactive user experience
- Simulate an e-commerce food delivery platform

 Key Features
- Attractive homepage layout
- Food categories and item listings
- Restaurant / food cards UI
- Responsive design for mobile and desktop
- Interactive elements using JavaScript
